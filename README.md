免费的Office AI工具

## 更新日志：
1，重构代码结构  
2，UI优化  
3，支持连续对话  
4，优化异常处理  
5，增加大模型配置  

## 使用说明：
1，通过本地Ollama部署大模型，在“FreeOfficeAI.Core”→“OllamaApi”中配置相关API  
2，通过VSTO集成到Word和Excel中  
3，Word支持总结提炼、合同检查、纠错矫正、翻译、对话功能  
4，Excel支持对话和公式(通过AI编写VBA自动处理文档)  

更多功能持续更新中...
